# 软件工程知识图谱个性化学习推荐系统

## 项目简介
本系统基于知识图谱技术，构建软件工程领域的知识点关系网络，实现个性化学习路径推荐。通过三维可视化界面，直观展示知识点之间的依赖关系，帮助用户理解软件工程知识体系结构。

## 核心功能

### 1. 知识图谱可视化
- 三维节点展示软件工程知识点
- 支持旋转、缩放、平移交互
- 节点颜色标识知识点难度等级
- 连线展示知识点间的前置依赖关系

### 2. 个性化学习推荐
- 根据用户学习进度推荐下一知识点
- 考虑知识点难度和用户掌握程度
- 生成可视化学习路径

### 3. 知识检索与探索
- 关键词搜索定位知识点
- 射线交互查看知识点详情
- 展示前置知识和后续知识

### 4. 学习进度管理
- 记录用户学习历史
- 知识点掌握状态标记
- 进度可视化展示
